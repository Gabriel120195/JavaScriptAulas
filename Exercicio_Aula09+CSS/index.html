História da Internet

O que é internet? [1.1]

Podemos dizer que a internet é um sistema global de redes computadores interligadas que utilizam um conjunto próprio de protocolos(Internet protocol suite ou TCP/IP) com o propósito de servir progressivamente usuários no mundo inteiro.

Creio que o termo internet mudou ao longo dos anos, com o avanço da tecnologia à internet passou a ser algo essencial para o ser humano, através dela consultamos o médico, nos comunicamos com pessoas do mundo todo, se tornou nossa principal fonte dê notícias. Podemos dizer que o mundo mudou completamente depois da criação da internet.

O que foi a ARPANET? [1.2]

A ARPANET (Advanced Research Projects Agency Network) foi uma rede de computadores criada em 1969 para transmissão de dados militares sigilosos é interligado com o departamento de pesquisa nos Estados Unidos.

Foi a primeira a fazer comutação de pacotes para computadores a distância e a implementar o protocolo de rede NCP e o protocolo TCP/IP, tornando-se base e início da rede mundial de computadores a internet.

Como surgiu a internet no Brasil? [1.3]

A Internet chegou no Brasil em 1988 por iniciativa da comunidade acadêmica de São Paulo (FAPESP - Fundação de Amparo à Pesquisa do Estado de São Paulo) e Rio de Janeiro UFRJ (Universidade Federal do Rio de Janeiro) e LNCC (Laboratório Nacional de Computação Científica).

No ano de 1989 foi criado o ministério da ciência e tecnologia, a rede nacional de pesquisas (RNP), que tinha como objetivo disponibilizar o acesso à internet nas instituições educacionais.Em dezembro de 1994 foi criado o projeto piloto da embratel para exploração comercial da internet através de linhas discadas.

O que é protocolo de internet? [1.4]

O protocolo de internet ou IP é usado para comunicação entre todas as máquinas em rede para encaminhamento dos dados, essa comunicação pode ser realizada pelos protocolos de comunicação TCP/IP ou OSI, o importante protocolo da internet IP está na camada intitulada camada de rede.

Todos os dados que são enviados e recebidos no IP, são referidos como ficheiros, no IP nenhuma definição é necessária antes do nó tentar enviar ficheiros para um nó com o qual não comunicou previamente. O IP oferece um pacote de datagramas (pacotes) no qual o pacote chegar sem garantias de conter os ficheiros corretos, podendo chegar desordenados ou duplicado, para aplicações que requerem mais confiabilidade, é adicionado uma camada de transporte.

Cite os principais crimes cibernéticos [1.5]

A definição mais simples de crime cibernético é atividade criminosa que tem como alvo ou faz uso de um computador, ou dispositivos conectados à rede. Os crimes podem ser divididos em duas categorias: visando os computadores e informações contidas ou utilizado de computadores para outros crimes.
Os tipos mais comuns são:

  • Roubo de dados financeiros ou relacionados a pagamento de cartões;
  • Roubo e venda de dados corporativos;
  • Fraude por e-mail e pela Internet;
  • Fraude de identidades, quando informações pessoais são roubadas e usadas;
  • Extorsão cibernética, que exige dinheiro para impedir o ataque ameaçado;

História da WEB

O que é a WEB? [2.1]

A Word Wide Web (WWW ou simplesmente Web), é um meio de comunicação global no qual os usuários podem ler e escrever através de computadores conectados à internet. Com o surgimento da Web foi quebrada uma barreira de como as empresas e pessoas se comunicam ou como compramos e vendemos qualquer tipo de produto.

A utilização do WWW pelo usuário, designa um sistema de documentos (ou hipermídia) que são interligados e executados na internet. A palavra Web é usada para se referir a internet ou estar conectado na rede de computadores.

Como e quando surgiu a WEB? [2.2]

O desenvolvimento da World Wide Web começou em 1980, quando o inglês Tim Berners-Lee, um funcionário contratado do CERN (Organização Europeia para a Investigação Nuclear), na Suíça, desenvolveu o ENQUIRE, um projeto usado para reconhecer e armazenar associações de informações. Cada nova página no ENQUIRE deveria estar ligada a uma página existente.

Nos anos seguintes Tim Berners-Lee foi desenvolvendo melhor sua ideias e paralelamente buscando colaboradores com recursos para colocar em prática o projeto de unir o hipertexto com a internet. Somente em dezembro de 1990, Lee conseguiu criar as ferramentas necessárias para funcionamento da Web, o protocolo de transferência de hipertexto (HTTP), a linguagem de marcação de hipertexto (HTML), o primeiro navegador (browser) chamado WordWideWeb e um servidor (CERN httpd).

Como funciona a WEB? [2.3]

Na Word Wide Web os documentos ou páginas Web estão interligados através de links que utilizam a internet para funcionar, utilizamos o navegador através das URLs acessamos as páginas ou aplicações desejadas.

De maneira mais técnica o usuário digita a URL no navegador acessando uma hiperligação, a parte da URL referente ao servidor é separada e transformada em um endereço IP, por um banco de dados chamado (DNS), o navegador estabelece então uma conexão TCP/IP com o servidor Web. Nos próximos passo o navegador envia uma requisição HTTP ao servidor para obter recursos adicionais da página e posteriormente o navegador renderiza a página na tela de usuário.

Desenvolvimento WEB + HTML

O que é e como surgiu HTML? [3.1]

HTML (HyperText Markup Language ou linguagem de marcação de texto) é uma linguagem de marcação utilizada na construção de páginas Web, os documentos HTML podem ser interpretados por navegadores. A tecnologia é fruto da junção de dois padrões HyTime e SGML.
HyTime é um padrão para representação estruturada de hipermídia e conteúdo baseado no tempo.
SGML é um padrão de formatação de textos, transforma documentos em hiper-objetos e para descrever ligações.

Tim Berners-Lee criou o HTML original e outros protocolos associados como HTTP, numa estação NexTcube que era um ambiente de desenvolvimento da época. As primeiras versões do HTML foram definidas com regras de sintaxes flexíveis, que ajudou a popularizar as publicações na Web pelo fato da sua simples compreensão. Com o passar dos anos e com o avanço das tecnologias, foram criadas versões melhores do HTML e já utilizamos a versão 5.2

O que é e como surgiu CSS? [3.2]

O CSS (Cascading Style Sheets) é um mecanismo para adicionar ( cores, espaçamento, fontes, etc) a uma página HTML. O desenvolvedor pode aplicar diretamente na página HTML usando as tags style, ou adicionando um link para um arquivo CSS que contém os estilos criados pelo desenvolvedor. Geralmente a prática de associar o link é mais utilizada, pois é mais fácil para qualquer tipo de mudança e manutenções da página HTML.

O CSS foi proposto pela primeira vez em outubro de 1994 por Hakon Lie, que queria facilitar a programação de sites, que na época era algo extremamente complexo, como criar uma tabela necessitava muitas linhas de código. Em 1995 o CSS1 foi desenvolvido pela W3C, um grupo de empresas de informática, foi quando a linguagem ganhou a popularidade entre os programadores e posteriormente criadas novas tags e comandos e se tornou algo indispensável para criar páginas na web.
